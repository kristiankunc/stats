# STATS
See how much you have done

# Installation
How can I install this beautiful program?
## Linux and MacOS
1. use this command (you have to have `odin 2026-03`):
```make```

2. all done!

## Windows
1. I have no way to test this, however, this _should_ work:
```odin build stats -o:size```

2. good luck!
